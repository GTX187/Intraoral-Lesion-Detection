# Intraoral-Lesion-Detection
This project focuses on the development of an automated intraoral lesion detection and classification system using clinical metadata and image data. The system is designed to assist in the identification of Oral Potentially Malignant Disorders (OPMDs).

Using the cleaned and processed metadata and ROI that is extracted from the images (with the help of masked R-CNN), we create a combined extensive dataset that is feeded into CNNs like ResNet50 and U-Net for training and classification.
